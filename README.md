# Vercel Deploy Plugin for Codex

Deploy to Vercel directly from Codex with a claimable preview.

## Installation

```bash
$skill-installer vercel-deploy and manage Vercel deployments
```

## Prerequisites

- None. Deployments are claimable and do not require authentication.

## Commands

| Command | Description |
|---------|-------------|
| `/vercel-deploy` | Deploy with a claimable preview |

## Skills

Skills are auto-invoked based on context:

- **vercel-deploy** - "deploy my app", "deploy and give me the link"
